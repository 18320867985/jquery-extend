默认轮播


           CSS: 

            /* 大框默认宽100% 设置高度 */
           .v-fade {
                height: 400px;
             }

            /*主体内容设置宽度 水平居中  */
            .v-fade-wrap {
                width: 1200px;
                margin: 0 auto;
            }

每一项 自定义动画

          htmml:
           <a href="#">
                <img class="_fade-img" src="./img/v-fade//2-slide.png" alt="自定义动画图片" />
           </a>
     

            CSS: 动画元素

            /*默认样式*/ 
            ._fade-img {
                      right:100px;
              }

            /*激活样式*/ 
             &.active ._fade-img {
                    display:inline-block;
                    opacity: 1;
                    animation: v-dafe-in 2s ease both;
              }

            /*动画*/
            @-webkit-keyframes v-dafe-in {
              0% {
                visibility: visible;
                opacity: 0;
                -webkit-transform: translateX(-1200px) rotateY(0deg) scale(0);
                        transform: translateX(-1200px) rotateY(0deg) scale(0); }
              100% {
                opacity: 1;
                -webkit-transform: translateX(0) rotateY(360deg);
                        transform: translateX(0) rotateY(360deg); } }

            @keyframes v-dafe-in {
              0% {
